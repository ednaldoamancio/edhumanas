# EdHumanas - Portal de Educação em Ciências Humanas 🌍🌱

Bem-vindo ao repositório do **EdHumanas**, um portal educacional focado em fornecer planejamentos de aula estruturados e de alta qualidade para as disciplinas da área de Ciências Humanas.

🔗 **Acesse o site:** [www.edhumanas.com](http://www.edhumanas.com) *(Em breve)*

## 🎯 Sobre o Projeto
O EdHumanas foi criado para auxiliar professores na organização e aplicação de metodologias de ensino conectadas à BNCC e aos principais vestibulares do país. O site disponibiliza roteiros práticos, dinâmicas e planos bimestrais com foco no desenvolvimento integral dos alunos.

### 📚 Planejamentos Disponíveis Atualmente:
* **Geografia (Ensino Médio - 1º, 2º e 3º Ano):** Foco em conceitos fundamentais, geopolítica, demografia e forte revisão para o ENEM e vestibulares (FUVEST, Unicamp, Unesp).
* **Projeto de Vida (Ensino Fundamental II - 6º Ano):** Roteiros focados em sondagem diagnóstica, autoconhecimento, inteligência emocional e desenvolvimento de competências socioemocionais.

## ✨ Funcionalidades
* **Design Responsivo:** Interface moderna, limpa e acessível em qualquer dispositivo (Celular, Tablet, Desktop).
* **Exportação de Documentos:** Todos os planos de aula possuem funcionalidades integradas para exportação com apenas um clique para:
    * 📄 PDF (usando a biblioteca `html2pdf.js`)
    * 📝 Microsoft Word (.doc)
* **Identidade Visual Institucional:** Cabeçalhos em tabelas adaptados para manter os logotipos governamentais e escolares perfeitamente alinhados na impressão.

## 🛠️ Tecnologias Utilizadas
* HTML5 (Semântico)
* CSS3 (Flexbox, Grid, Variáveis e UI Moderna)
* JavaScript Vanilla
* [html2pdf.js](https://github.com/eKoopmans/html2pdf.js) - Para renderização dos PDFs direto no navegador.
